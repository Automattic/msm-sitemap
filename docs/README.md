## Quick Start

1. **Installation**: See the [main README](../README.md) for installation instructions
2. **Basic Usage**: Use the admin interface at **Settings > Sitemap**
3. **CLI Management**: Use `wp msm-sitemap` commands for advanced management
4. **API Integration**: Use the REST API for programmatic access
5. **Customization**: Use hooks and filters to customize behavior
